AEM Asset Selector

Welcome to AEM Asset Selector

Browse your AEM DAM and drag-drop assets directly into Adobe Target, Salesforce, Braze, and any web app. Let's get you connected in 3 steps.

What you'll need

Your Adobe IMS credentials — imsClientId, imsOrg, and apiKey. If you don't have these yet, we'll help you generate the support ticket to get them.

  • AEM as a Cloud Service access
  • Admin Console access (for the support ticket)
  • Chrome browser with the extension installed

Enter your AEM credentials

These are stored locally in your browser only — never sent to our servers.

Found in Adobe Admin Console → Organization profile
Provided by Adobe after Asset Selector provisioning
Often the same as Client ID for many Adobe integrations
Your AEM author instance URL — limits browsing to this repo

Request Adobe provisioning

You need to file a P2 support ticket with Adobe to get your imsClientId. Fill in the details below and we'll generate the ticket text for you.

Your AEM as a Cloud Service author environment URL

Important: Redirect URL

Make sure the ticket includes assetselector.vyingdigital.com as the domain. Adobe will add this to your IMS redirect URL allowlist, which is required for authentication.

You're all set!

Your AEM Asset Selector is loading. Browse your DAM, select assets, and drag them into any web app.

How to use

1. Browse and search for assets in the selector above.
2. Select assets — they'll appear in the tray at the bottom.
3. Drag any asset from the tray into the active web page.
4. Drop zones in Target, Salesforce, Braze etc. light up automatically.
